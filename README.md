# An Simple React Native Application for searching Houses or Rooms in Cities, Towns, etc..

<h1><u>Inorder to run the Code:</u></h1>
<ul>
<li>[1] git clone https://github.com/Daniel-V-Richardson/InstaRental.git</li>
<li>[2] npm install</li>
<li>[3] npm start</li>
</ul>
